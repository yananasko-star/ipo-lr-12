# Задание 1: Класс Client

## Описание задания
**Создать класс Client для клиентов компании.**

### Требования к классу:

#### Атрибуты:
- name – имя клиента
- cargo_weight – вес груза
- is_vip – флаг VIP-статуса (по умолчанию False)

#### Важные требования:
1. Все методы должны иметь валидацию данных
2. Класс должен находиться в пакете transport
3. Класс должен быть в отдельном модуле client.py

### Пример использования:
`python
from task_1.transport.client import Client

# Создание обычного клиента
client1 = Client("Иван Иванов", 5.5)
print(client1)  
# Вывод: Клиент: Иван Иванов, Груз: 5.5т, Статус: обычный

# Создание VIP клиента
client2 = Client("Петр Петров", 10.0, True)
print(client2)  
# Вывод: Клиент: Петр Петров, Груз: 10.0т, Статус: VIP

# Проверка валидации
try:
    client3 = Client("", 5.0)  # Ошибка: пустое имя
except ValueError as e:
    print(f"Ошибка: {e}")

try:
    client4 = Client("Анна", -5.0)  # Ошибка: отрицательный вес
except ValueError as e:
    print(f"Ошибка: {e}")
